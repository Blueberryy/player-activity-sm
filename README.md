# Time played
This plugin will display statistics of clients time through sm_time command.

# Database configuration
```
"timestats" 
{ 
    "driver"            "mysql" 
    "host"                "" 
    "database"            "" 
    "user"                "" 
    "pass"                "" 
    //"timeout"            "0" 
    //"port"            "0" 
} 
```

# Example of output
```
12 minutes - current session
22 hours - past two weeks
335 hours - on record
```

# Players activity
This plugin will display informations of players activity on the server.

# Database configuration
```
"activity" 
{ 
    "driver"            "mysql" 
    "host"                "" 
    "database"            "" 
    "user"                "" 
    "pass"                "" 
    //"timeout"            "0" 
    //"port"            "0" 
} 
```

# Example of output
```
12 minutes - current session
22 hours - past two weeks
335 hours - on record
```
